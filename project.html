Rock-Paper-Scissor

The Rock-Paper-Scissors game is a simple web application created using HTML, CSS, and JavaScript. It allows users to play the classic game against the computer. The interface is built with HTML and styled with CSS, while JavaScript handles the game logic. Players choose rock, paper, or scissors, and the computer randomly selects its move. The winner is determined based on traditional rules, and the result is displayed. This project highlights basic web development skills, including responsive design, and serves as an example of creating interactive applications with fundamental web technologies.
